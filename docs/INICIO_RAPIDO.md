# 🚀 INÍCIO RÁPIDO - Sistema de Loja de Informática

## 3 Passos Simples

### 1️⃣ Instalar Dependências
```bash
pip install -r requirements.txt
```

### 2️⃣ Configurar Base de Dados
```bash
python setup_db.py
```
Cria BD, tabelas e utilizadores de teste.

### 3️⃣ Executar Aplicação
```bash
python main.py
```
