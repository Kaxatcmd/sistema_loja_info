# Referência de Ícones - Sistema de Loja

## Mapear Antigo → Novo

Esta documentação define os ícones/símbolos utilizados no sistema para manter consistência profissional.

### Estrutura & Operações Gerais
- 🏪 → [LOJA] - Identificação do sistema
- ✓ → [✓] - Sucesso / Confirmado
- ✗ → [✗] - Erro / Falha
- ✅ → [OK] - Operação completada com sucesso
- ❌ → [ERRO] - Falha na operação

### Segurança & Autenticação
- 👤 → [USER] - Utilizador/Cliente
- 👨‍💼 → [ADMIN] - Administrador
- 🔐 → [LOCK] - Segurança/Logout
- 🔧 → [CONFIG] - Configuração/Ferramentas
- 🔍 → [FIND] - Busca/Pesquisa

### Gestão de Produtos
- 📦 → [ITEM] - Produto/Item
- 🛍️ → [STORE] - Compras/Explorar
- 🛒 → [CART] - Carrinho de compras
- ➕ → [ADD] - Adicionar item
- 📊 → [STATS] - Estatísticas/Relatório

### Gestão de Dados
- 📝 → [FORM] - Formulário/Criação
- 📋 → [LIST] - Listagem de dados
- 📥 → [IMPORT] - Importar/Adicionar dados
- 💰 → [SALES] - Vendas/Valores
- 💡 → [TIP] - Dica/Informação

### Status & Feedback
- ℹ️ → [INFO] - Informação
- ⚡ → [QUICK] - Rápido/Otimizado
- 🚀 → [START] - Inicializar/Começar
- 📧 → [EMAIL] - Email/Comunicação
- 🎉 → [DONE] - Pronto/Concluído

---

## Estilos de Uso

### Títulos Principais
```
[LOJA] SISTEMA DE INFORMÁTICA
```

### Mensagens de Status
```
[OK] Arquivo guardado com sucesso
[ERRO] Conexão falhada
[✓] Tarefa concluída
[✗] Tarefa cancelada
```

### Menus & Botões
```
[STORE] Explorar Produtos
[CART] Ver Carrinho
[ADMIN] Painel Admin
[ITEM] Gerir Produtos
```
