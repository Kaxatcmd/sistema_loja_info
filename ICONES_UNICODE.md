# Mapa de Ícones Unicode Minimalistas - Sistema de Loja

## Símbolos Utilizados

Estes são símbolos Unicode minimalistas e profissionais em preto e branco, sem parecer infantil.

### Estrutura & Operações Gerais
- 🏪 → ⌂ (Casa/Loja)
- 📊 → ▣ (Estatísticas)
- ✓ → ✔ (Sucesso/Confirmado)
- ✗ → ✘ (Erro/Falha)
- ✅ → ✓ (Operação concluída)
- ❌ → ✗ (Falha na operação)

### Segurança & Autenticação
- 👤 → ◦ (Utilizador)
- 👨‍💼 → ▲ (Administrador/Superior)
- 🔐 → ◆ (Segurança/Bloqueado)
- 🔧 → ⚙ (Configuração/Ferramentas)

### Gestão de Produtos & Compras
- 📦 → ▬ (Produto/Caixa)
- 🛍️ → ▸ (Explorar/Procurar)
- 🛒 → ▪ (Carrinho/Item)
- ➕ → ⊕ (Adicionar)
- 💰 → ◈ (Vendas/Valores)

### Gestão de Dados
- 📝 → ▥ (Formulário/Linhas)
- 📋 → ▨ (Listagem)
- 📥 → ⬇ (Importar)
- ℹ️ → ⓘ (Informação)
- ⚡ → ★ (Rápido/Destaque)

### Status & Feedback
- 🎉 → ◇ (Pronto/Celebração)
- 🚀 → ► (Inicializar)
- 📧 → ◉ (Email/Contacto)
- 💡 → ◐ (Dica/Luz)

## Exemplos na Interface

### Terminal/Console
```
⌂ LOJA DE INFORMÁTICA
▣ Verificando base de dados...
✔ Base de dados pronta!
▥ Verificando tabela 'clientes'...
✔ Tabela criada!

▲ Administrador
◦ Utilizador
⚙ Configuração
```

### Interface Gráfica (Tkinter)
```
[⌂] Sistema de Loja de Informática

[◦] Bem-vindo, João!
[◆] Logout

Botões:
[▸] Explorar Produtos
[▪] Ver Carrinho
[▲] Painel Admin
[▬] Gerir Produtos
[⊕] Novo Produto
```

---

**Nota:** Todos os símbolos são caracteres Unicode padrão, compatíveis com qualquer terminal e editor de texto.
